Comportement sur image rejetée

Quand la galerie reçoit une photo de famille ou screenshot — que se passe-t-il ?

A

Blocage strict

Bouton "Analyser" désactivé. Message sous l'image : "Image non reconnue comme carte de visite". L'utilisateur doit reprendre ou choisir une autre image.

Pour

  • Zéro appels inutiles au modèle
  • Feedback immédiat et clair

Contre

  • Bloque si faux négatif (carte mal détectée)
B

Avertissement + forcer

Badge d'avertissement sur l'image. Bouton "Analyser quand même" disponible. L'utilisateur garde le choix final.

Pour

  • Pas bloquant si faux négatif
  • Éduque sans empêcher

Contre

  • Les utilisateurs cliquent "forcer" par réflexe
C

Seuil permissif

Filtre très tolérant — bloque uniquement les cas évidents (selfie, paysage plein écran). Doute = laisse passer.

Pour

  • Presque jamais de faux négatifs
  • Expérience fluide

Contre

  • Filtre moins efficace, économie moindre
