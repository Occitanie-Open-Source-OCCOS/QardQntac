Quel type d'organisation ?

Les contacts existent déjà — les dossiers viennent s'y ajouter

A

Dossiers plats

Dossiers simples sans hiérarchie. Ex: "Travail", "Famille", "Événement BizDay". Un contact = un dossier.

B

Tags / étiquettes

Un contact peut appartenir à plusieurs catégories simultanément. Ex: Jean Dupont → "Client" + "Paris" + "Lead". Plus flexible que les dossiers.

C

Dossiers hiérarchiques

Dossiers avec sous-dossiers. Ex: Travail → Clients → France. Plus puissant mais plus complexe à gérer.
